#  Mi Portafolio Web

##  Características de la página

Esta página fue pensada para crearse como un modelo serio para lucir en mi perfil de GitHub. 

Está estructurada en su página principal (`index.html`), la página de presentación (`presentacion.html`) y un unico archivo de css (`styles.css`).

Es una página simple creada con el propósito de testear mi capacidad para hacer sitios **responsive**, dividiendo la presentación en dos cuadros de la misma proporción. También utiliza elementos fundamentales como:

* **Botones interactivos:** Enlaces decorados con CSS con apariencia de botones (contienen animacioon hecha en css con hover y transform).
* **Flexbox:** Para la adaptabilidad y maquetación en diferentes dispositivos.
* **Estructura HTML limpia:** Uso correcto de etiquetas y organización (cabe aclarar que hice un abuso de ids en mi proyecto debido a la falta de JavaScript y el testeo de la misma).

## 🤖 Uso de Inteligencia Artificial

> [!NOTE]

Esta página fue creada con el fin de testear el uso de **Flexbox, listas y enlaces**, además de transmitir mis metas y las razones de por qué hago esto.
